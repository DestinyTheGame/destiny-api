# destiny-api

A module that interact's with Bungie's Destiny API. Written in JavaScript so it
can be used to build applications using:

- Browser plugins
- React-Native
- PhoneGap
- Titanium Appcelerator
- Node.js / IO.js

## Installation

```
npm install --save destiny-api
```

## Usage

```js
var destiny = new Destiny({ /* options */ });
```
